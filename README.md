# CI/CD Pipeline with Azure DevOps (.NET)

Mise en place d’un pipeline d’intégration continue (CI) pour une application .NET avec Azure DevOps.

## Fonctionnalités
- Gestion du code avec Git (branches, pull-requests)
- Pipeline CI en mode Classic et YAML
- Build automatique des projets .NET
- Génération d’artefacts
- Déclenchement automatique sur les branches main et develop

## Technologies
- Azure DevOps
- Git
- YAML
- .NET 8
- CLI (dotnet)

## Objectif
Automatiser le processus de build et améliorer la qualité logicielle via l’intégration continue.
